## 📖 Overview

FinTrack is a responsive finance management application that allows users to record transactions, monitor spending, and visualize financial data through interactive charts.

The application stores data locally using browser Local Storage, ensuring that transaction history persists even after refreshing the page.

---

## ✨ Features

### Transaction Management

* Add Income and Expense transactions
* Record Amount, Category, Date, and Notes
* Delete transactions
* Automatic validation for user inputs

### Financial Summary

* Total Income
* Total Expenses
* Net Balance
* Savings Rate
* Top Spending Category

### Analytics

* Interactive Pie Chart showing spending by category
* Smart rule-based financial insights

### Filtering

* Filter transactions by category

### User Experience

* Responsive design
* Modern fintech-inspired UI
* Automatic data persistence using Local Storage
* Clean and intuitive interface

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Vinay171105/fintrack.git
```

Navigate into the project:

```bash
cd fintrack
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
